Features

WireGuard VPN

Auto-connects when you open the app. All traffic is end-to-end encrypted before it leaves your device.

Persistent Sessions

tmux keeps sessions running on your server. Lock your phone, switch to iPad — pick up exactly where you left off.

Claude Code

Native chat UI with streaming responses, tool use chips, and permission approval — all over your own Bedrock endpoint.

Full Terminal

SwiftTerm-powered xterm emulator for shell, Vim, Kiro, or anything that runs in a terminal.

QR Onboarding

Scan one QR code to configure WireGuard, auth token, and server endpoints in under 10 seconds.

Push Notifications

Get notified when a Claude Code session finishes, even when the app is in the background.

How it works

1

Deploy your server

Run ./deploy.sh create local --mobile-app directly on your Mac or Linux machine. Or SSH into any Ubuntu server with create any, or spin up a new VM on Lightsail, EC2, Hetzner, or DigitalOcean.

2

Scan the QR code

The deploy script prints a one-time QR code containing your WireGuard config and auth token. Scan it with the app — setup is complete.

3

Open sessions anywhere

Create Claude Code, Kiro, or terminal sessions from the session list. They keep running on your server regardless of your connection.

4

Never lose context

Come back hours later on a different device. Sessions are exactly as you left them — no reconnect friction, no lost work.

Your data stays yours

Continuum connects only to your server over an encrypted WireGuard tunnel. No data is collected, no analytics run, no third-party servers are contacted. Read the full privacy policy →
